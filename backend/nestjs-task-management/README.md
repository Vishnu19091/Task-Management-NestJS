## Description

Task Management Backend API logic.

## Project setup

```bash
$ npm install
```

## Local Usage

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Deployment

Using [Tailscale](https://tailscale.com/kb/1223/funnel) and [Docker](https://docs.docker.com/get-started/) for deployment.
